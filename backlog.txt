Le Backlog Product


Les fonctionnalités:
- Le labyrinthe est généré à partir d'un fichier, en fonction du niveau sélectionné.
- Le labyrinthe contient des murs.
- Le héros est généré et initialement placé sur la case de départ.
- Le héros a un nombre initial de points de vie.
- Le héros doit pouvoir se déplacer à l’intérieur du labyrinthe, vers le haut, le bas, la droite et   la gauche. Il ne peut pas traverser les murs.
- Il existe des cases spéciales du labyrinthe : trésor, pièges, magiques, passages. Les effets qui se déclenchent lorsqu’un personnage marche dessus sont les suivants:
- Case trésor : le héros gagne s'il arrive sur cette case.
- Case pièges : le personnage subit des dégâts s'il arrive sur cette case.
- Case magique : un effet est déclenché si un personnage arrive sur cette case.
- Case passages : le personnage est téléporté à un autre endroit de manière aléatoire s'il arrive sur cette case.
- Les monstres sont générés et placés d'une manière aléatoire dans le labyrinthe.
- Les fantômes, qui sont des monstres pouvant traverser les murs, sont générés et placés d'une manière aléatoire dans le labyrinthe.
- Les monstres ont un nombre initial de points de vie. 
- Le déplacement des monstres se fait d'une manière aléatoire ou bien d'une manière intelligente pour attraper le héros. Une fois qu’il s’approche du héros, son déplacement change de telle sorte à ce qu’il attrape le héros (poursuite), c’est ce qu’on entend par “déplacement intelligent”.
- Le héros perd des points de vie lors d'un contact avec un monstre.
- Le héros peut attaquer le monstre avec lequel il est en contact ou sur la case adjacente. 
- Le héros peut récupérer des armes.
- Le héros attaque les monstres avec des armes. 
- Les monstres perdent des points de vie s’ils sont en contact avec le héros ou si le héros les attaque.
- Le héros meurt s'il perd tous ses points de vie.
- Un monstre meurt s’il perd tous ses points de vie. Dans ce cas, il disparaît de l’écran.
- Le héros gagne des points de vie s'il tue un monstre.
